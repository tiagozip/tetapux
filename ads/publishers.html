For publishers

In order to make Tetapux as easy to add as possible, you'll just need to import the Tetapux Ads library: 

<script src="https://cdn.jsdelivr.net/gh/tiagorangel1/tetapux/aw.min.js"></script>

Then, you can use the <tetapux-ads></tetapux-ads> custom component. The width and height parameters are supported, along with vertical to make the ad vertical and more coming soon. You can also specify custom HTML to inject into the shadow DOM by using inject="your code"
